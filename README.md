# جموع التكسير

جدولٌ يربط أوزان المفرد بأوزان جمع التكسير في العربية، مع الأمثلة، وتمييزٍ بين الاسم والصفة. مُستخرَجٌ من كتاب [دليل جموع التكسير](sources/%D8%AF%D9%84%D9%8A%D9%84%20%D8%AC%D9%85%D9%88%D8%B9%20%D8%A7%D9%84%D8%AA%D9%83%D8%B3%D9%8A%D8%B1.pdf) (مُعرف ويكي بيانات: [Q140078098](https://www.wikidata.org/wiki/Q140078098)) ومُراجَعٌ ومُدقَّق. يضمّ **557** تقابُلًا بين **143** وزنًا للمفرد و**61** وزنًا للجمع، مع **1782** زوجًا من الأمثلة (مفرد وجمعه).

🔗 **الموقع التفاعلي:** <https://forzagreen.github.io/ar-plurals>
