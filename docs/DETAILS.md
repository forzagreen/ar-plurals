# التفاصيل التقنية

تفاصيل بنية البيانات، وحقول الملفّات، وعروض الموقع، وإعادة البناء، والنشر. للملخّص والتنزيل انظر [README](../README.md).

## بنية البيانات

كل سطرٍ تقابُلٌ واحد: (وزن مفرد ← وزن جمع ← نوع). الأعمدة:

| العمود | الوصف |
|---|---|
| `المفرد` | وزن المفرد (مثل: `فَعِيل`) |
| `النوع` | `اسم` أو `صفة` |
| `الجمع` | وزن جمع التكسير (مثل: `فُعَلاء`) |
| `الأمثلة` | الأمثلة نصًّا، مع شروطها والشواذّ |
| `أمثلة` | *(في JSON فقط)* الأمثلة مُحلَّلة إلى بنية مُنظَّمة (انظر أدناه) |

المجموع: **557** تقابُلًا تربط **143** وزن مفرد بـ**61** وزن جمع، وفيها **1782** زوج مثال.

### الأمثلة المُنظَّمة (`أمثلة`)

في صيغة JSON تُحلَّل كل خليّة أمثلة إلى قائمة «كُتَل»، لكل كُتلة شرطُها وأزواجها، ويُميَّز الشاذّ:

</div>

```json
{
  "المفرد": "فاعِل", "النوع": "صفة", "الجمع": "فُعّال",
  "أمثلة": [
    { "شرط": "الصحيح اللام، للعاقل", "شاذ": false,
      "أزواج": [["قارِئ","قُرّاء"], ["كاتِب","كُتّاب"], ["جاهِل","جُهّال"]] },
    { "شرط": "", "شاذ": true,
      "أزواج": [["غازٍ","غُزّاء"], ["سارٍ","سُرّاء"]] }
  ]
}
```

<div dir="rtl">

| الحقل | المعنى |
|---|---|
| `شرط` | شرط التقابُل أو وصفه (قد يكون فارغًا) |
| `شاذ` | `true` إذا كانت الكتلة من الشواذّ (`وشذّ`) |
| `أزواج` | أزواج `[المفرد, الجمع]` من الأمثلة |
| `ملاحظة` | نصٌّ تفسيري إضافي *(اختياري)* |

## الملفّات والصِّيَغ

```
data/
├── csv/   جموع التكسير.csv     (UTF-8 BOM)
├── json/  جموع التكسير.json    (مع الأمثلة المُنظَّمة)
└── xlsx/  جموع التكسير.xlsx    (مُنسَّق، اتجاه RTL)
site/      موقع GitHub Pages الثابت (جدول، مخطّط سانكي، مصفوفة)
scripts/   نصوص البناء (Python)
sources/   المصدر: «دليل جموع التكسير.pdf»
```

## عروض الموقع

1. **الجدول** — بحثٌ فوريّ (يتجاهل التشكيل) وتصفية بالنوع، مع ثلاثة أنماط: الجدول الكامل، ومن المفرد إلى الجمع، ومن الجمع إلى المفرد. الأمثلة معروضةٌ أزواجًا (مفرد ← جمع) مع شروطها والشواذّ. النقر على أي وزن يعرضه في الجدول.
2. **مخطّط سانكي** — يعرض جميع التقابُلات؛ أوزان المفرد يمينًا وأوزان الجمع يسارًا. تمرير المؤشر على وزنٍ يُبرز ارتباطاته وحدها، وعلى خيطٍ يُظهر أمثلته.
3. **المصفوفة** — خريطةٌ حرارية كاملة (صفوف = المفرد، أعمدة = الجمع) تُبرز أكثر الأوزان إنتاجًا؛ تمرير المؤشر على خليّة يُظهر أمثلتها.

الموقع مكتفٍ ذاتيًّا (الخطوط وECharts محليًّا) ويعمل دون اتصال بالإنترنت.

## إعادة بناء البيانات

</div>

```bash
python -m venv .venv && source .venv/bin/activate
pip install openpyxl
python scripts/build_dataset.py   # يقرأ الجدول المُراجَع ويُنتج data/ + site/assets/data.js
```

<div dir="rtl">

- `scripts/build_dataset.py` — يقرأ الجدول المُراجَع، ويحذف عمود «الفروق»، ويُحلِّل الأمثلة، ويُخرج الصِّيَغ الثلاث وبيانات الموقع.
- `scripts/parse_examples.py` — مُحلِّل الأمثلة إلى بنية مُنظَّمة.
- `scripts/fetch_assets.py` — يُنزِّل ECharts وخطّ Noto Naskh Arabic محليًّا ليعمل الموقع دون اتصال.
- `scripts/serve.py` — خادمٌ محلّيٌّ للمعاينة يُعطِّل التخزين المؤقّت: `python scripts/serve.py`.
- `scripts/extract_tables.py` و`merge_tables.py` — نصوص الاستخراج والدمج الأصلية (للأرشيف).

## نشر GitHub Pages

ادفع إلى `main`؛ يتكفّل سير العمل في `.github/workflows/pages.yml` بنشر مجلّد `site/` (مع نسخة من `data/` للتنزيل). فعِّل **Settings → Pages → Source: GitHub Actions**.
